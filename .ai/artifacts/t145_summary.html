02 Phases

A strict sequence — each phase ended green on cargo build plus its own gate before the next began. The G5 differential harness ran continuously from Phase 6 onward.

0

Scaffold + CI swap

Cargo workspace (axum 0.8, sqlx 0.9, tokio, jsonwebtoken 10, jsonschema, reqwest+rustls-ring, ammonia, governor), toolchain pin 1.95, minimal bin/api.rs, additive rust-backend CI job + rust-* make targets.

1

DB schema & migrations G2

Schema frozen from pg_dump --schema-only of a Go-migrated reference DB, iterated until the diff was empty. Connect + backoff + tuned pool; refresh_leaderboard.

2

Models + encoder contract

29 structs / 12 enums, field order = Go. The anti-regression core: go_time serializer, omitempty via skip_serializing_if, float8 casts, jsonb passthrough, time_of_day::text, 4 soft-delete tables.

3

Config + boot

16 env vars (hard-fail on two), boot order mirroring Go, graceful SIGINT/SIGTERM shutdown, real client IP for rate-limiting.

4

Middleware

request-id · logging · panic recovery · CORS (exact allow-list, never *) · rate-limit (per-IP global + strict prefixes) · body-limit (1 MB, version POST → 256 MB) · tier extractors preserving mission_maker > leader.

5

Realtime SSE hub

tokio::broadcast per topic; two SSE endpoints (server-status = hub, audit = DB-poll) preserving frame format + X-Accel-Buffering.

6

Auth vertical G7a

HS256 JWT, Discord OAuth2, and the rotating single-use refresh invariant verbatim: reuse-of-revoked ⇒ revoke family + 401; atomic conditional UPDATE with rows_affected != 1 ⇒ reuse.

7

Contract & codegen G4

Rust target added to codegen.mjs (quicktype serde + rustfmt); two runtime validators re-implemented with jsonschema; schemas reached via include_str!.

8

Services G6 · G8

The crown jewel flatten_to_mod_document (compiled mod doc, sorted keys, locked coord map, NoSlots) + ORBAT parse/derive, mortar, webhook, discord, role sync, audit, purge, sanitizer.

9

Handlers — 84 routes G1 · G5

Every route, every envelope + status code. Heaviest: events (ORBAT + the slot-claim race G7b), missions (413/409, /compiled live), telemetry (idempotent ingest + MV refresh + SSE publish — closes the live loop).

10

Tests — 68 → 76 G3

Every Go test mapped; mock-HTTP webhook/discord suites, the concurrency race, archive/soft-delete lifecycle, body-limit override, token purge. 0 failed.

11

Standards + differential + cutover G5

G5 harness (Go vs Rust, seeded, ≡ 18/18), make rust-ci green, then the cutover — 85 .go files + module + Go verify scripts removed; CI / Makefile / codegen rewired to Rust.

03 Encoder contract

Go's encoding/json ≠ serde. Equivalence is canonical JSON equality, not byte-identity — each hazard has an exact resolution.

HazardResolution
§5 timestamps44 fields — Go RFC3339Nano trims trailing-zero nanos (.5 not .500) → custom go_time serializer
§5b dates2 date cols emit full …T00:00:00Z → decode as midnight DateTime<Utc>
§6 omitempty~60 fields → Option<T> + skip_serializing_if; empty string → str::is_empty
§7 numeric6 cols — sqlx decodes numeric→Decimal, never f64 → CAST … AS float8
§8 jsonb2 cols pass Postgres bytes verbatim → Json<Box<RawValue>> passthrough
§9 time_of_daytime column, Go field string → SELECT time_of_day::text
inet ipGo emits bare host; ::text keeps /32 → host(ip)
